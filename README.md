# 90s J-Pop Acoustic Profile Project

### Why does 90s J-Pop sometimes sound disappointing on high-end audio systems?
Many listeners who grew up with 1990s Japanese pop and rock remember these albums sounding energetic and vocal-forward. However, when played today on highly revealing DACs, amplifiers, and speakers, the same albums can sound surprisingly distant or restrained.

This project is an attempt to recreate that original listening experience using modern digital EQ. For nearly 30 years, I assumed that 1990s J-Pop simply didn't sound good on high-end audio systems. I thought this was an unavoidable limitation of the recordings themselves. I now believe that assumption was only partly true.

This profile is not intended to be objectively correct. It is a listening profile developed to reproduce a familiar 1990s listening experience on a modern high-fidelity playback system.

### Philosophy
This project does not attempt to "correct" old recordings.

Instead, it aims to recreate the listening experience that many people remember from the 1990s.

### Usage
1. Install foobar2000.
2. Download the latest version of 90s-JPOP-Acoustic-Profile-v**.feq.
3. In foobar2000->Settings->DSP Manager, add Equalizer.
4. In Equalizer, load the 90s-JPOP-Acoustic-Profile-v**.feq.
